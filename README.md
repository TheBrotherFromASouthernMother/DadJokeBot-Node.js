## Dad Joke Bot

Dad Joke Bot is a simple servless application that delivers a random dad joke to a receptient at a set time.


## Technologies used
  * JavaScript (ES6)
  * Node.js
  * Cron package
  * Nexmo Api
  * Git
  * Heroku
